# Assets

## sport-banner.jpg — foto del banner SPORT (home)

Carica qui un file chiamato **esattamente** `sport-banner.jpg` per usarlo come
sfondo del banner SPORT sotto i Fresh Drops.

- Nome file: `sport-banner.jpg` (tutto minuscolo)
- Formato consigliato: JPG orizzontale, largo (es. 1920×800 px o simile)
- Soggetto: runner / atleta / prodotto sportivo, meglio se scuro/notturno
  (il banner applica già un overlay scuro per la leggibilità del testo)
- Preferibilmente **senza testo** già impresso: la scritta "DO IT." viene
  aggiunta dal sito. Se la foto ha già una scritta, va bene lo stesso ma
  potrebbe sovrapporsi.

Appena il file è presente e il sito è ripubblicato, il banner lo usa in
automatico. Finché non c'è, il banner usa la foto di un prodotto sportivo.
